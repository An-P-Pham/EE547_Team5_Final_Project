Necessary dependencies: Before running our application, you will need the following dependencies
python3: version 3.9.2
python3-pip3: download the following packages (pandas, numpy, boto3, json, matplotlib, and sklearn)
docker: start mongoDB with the following command: "docker run -d -p 27017:27017 --name mongo mongo:5"
node: install node 
npm: run "npm install" to install all of our node modules
run our application locally: "node app.js"
